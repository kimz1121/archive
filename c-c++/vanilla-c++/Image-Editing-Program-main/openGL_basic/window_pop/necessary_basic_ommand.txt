chmod : 파일 권한 변경명령어 (현재 775(rwxrwxr-x)로 작성 및 사용 중)
gcc : 컴파일 명령어
ls : 파일 상태 확인 권한(퍼미션)등을 확인 할 수 있음 파일 실행 권한 여부

사용하는 라이브러리 설치사항
sudo apt install libsdl2-dev

사용하는 라이브러리 단축이름
-lSDL2
-lglfw

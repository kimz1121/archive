1. 영상으로 부터 이미지 자료 추출
    -파일 경로 지정
    -이미지당 사진 간격 지정

2. 레이블 링프로그램을 통한 픽셀 단위 상태좌표 자료 작성
    -레이블링 프로그램 사용전 predefined_classes.txt 파일의 내용 작성
        r1
        r2
        r3
        r4
        e0
        y_+_axis_vector
        y_-_axis_vector

3. 기준점의 절대 좌표 정리 자료 작성
    -레이블링 을 진행함과 동시에 
        y_+_axis_vector
        y_-_axis_vector
        위 두벡터의 공통 시점이 되는 상대 좌표계의 원점의
        절대좌표계상 좌표를 직접 적어 정리
        이때 파일 양식은 프로그램이 읽을 수 있도록 아래의 예시 설명을 따른 CSV파일로 정리

    -예시 자료
        -파일 경로 : 
            > 이미지 좌표 계산/3. 데이터 분석 프로그램/absolute_origin_list.csv 
            또는 
            > 이미지 좌표 계산/0. 예시 데이터/absolute_origin_list.csv

        -파일 작성 형식 예시
            x,y
            0,0.45
            0,0.45
            .
            .
            .
            7.2,1.35
            7.2,1.35

4. 데이터 분석 프로그램 사용
    -자료 준비
        데이터 분석 프로그램을 이용하기 위해서는 아래의 입력 자료가 필요하다
            -레이블링 프로그램을 통한 픽셀 단위 상태좌표 자료
            -기준점의 절대 좌표 정리 파일
            
            두 파일을 absolute_position_clac_01.py 프로그램이 접근할 수 있는 위치에 정리한다.

    -파이썬 프로그램 조작
        파이썬 프로그램을 편집기로 열어 입력할 자료들의 위치를 자신의 상황에 알맞게 수정한다.
        <추천> : 편집하기 전에 제공된 상태 그대로의 프로그램을 작동시켜 예시 프로그램의 동작을 확인하면 도움이 될 것.

        -수정 대상
            -"레이블링 프로그램을 통한 픽셀 단위 상태좌표 자료"의 입력 경로
            -"기준점의 절대 좌표 정리 파일"의 입력 경로
            -"출력 파일"의 이름
